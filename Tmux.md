
# tmux Shortcut Map

**Prefix key:** `Ctrl + A`

## Window Management

| Shortcut                | Action                       |
| ----------------------- | ---------------------------- |
| `Prefix + number (1–9)` | Switch to window             |
| `Alt + number (1–9)`    | Switch to window (no prefix) |
| `Prefix Shift + r`      | Reload tmux config           |
| `Prefix Alt + c`        | New window                   |

---

## Pane Splitting

| Shortcut       | Action                               |
|---------------|--------------------------------------|
| `Prefix + |`  | Split pane vertically (side-by-side)|
| `Prefix + _`  | Split pane horizontally (top/bottom)|

---

## Pane Navigation

| Shortcut     | Action              |
| ------------ | ------------------- |
| `Prefix + h` | Move to left pane   |
| `Prefix + j` | Move to bottom pane |
| `Prefix + k` | Move to top pane    |
| `Prefix + l` | Move to right pane  |
